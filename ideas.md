# Stratégie de Design - Portfolio SIG & Data

## Analyse du site de référence (naledi.co.uk)

Le site de Naledi Hollbruegge est un excellent modèle de portfolio minimaliste pour un professionnel du data. Caractéristiques clés :

- **Typographie claire** : Hiérarchie nette avec titres forts et corps lisible
- **Espace blanc généreux** : Respiration entre les sections
- **Minimalisme sobre** : Pas de décoration inutile, focus sur le contenu
- **Palette neutre** : Fond blanc/gris clair, texte noir/gris foncé
- **Structure linéaire** : Navigation simple (Home, Blog)
- **Sections thématiques** : Collect, Process, Visualise (3 piliers)
- **Portfolio visuel** : Projets avec descriptions concises
- **Appel à l'action clair** : "GET IN TOUCH"
- **Testimonials** : Preuve sociale
- **Responsive** : Adapté mobile et desktop

## Trois approches de design

### Approche 1 : Minimalisme Épuré (Probabilité : 0.08)

**Design Movement** : Swiss Minimalism / Modernisme Rationnel

**Core Principles** :
1. Grille invisible mais rigoureuse
2. Typographie sans-serif exclusive (Helvetica, Inter)
3. Espace blanc comme élément de design
4. Pas de couleur, uniquement noir/blanc/gris

**Color Philosophy** : 
- Fond blanc pur (#FFFFFF)
- Texte noir profond (#1A1A1A)
- Accents gris neutre (#666666)
- Pas de couleur secondaire

**Layout Paradigm** : 
- Colonnes strictes
- Alignement parfait
- Marges équilibrées
- Sections empilées verticalement

**Signature Elements** :
- Lignes horizontales fines
- Espacements réguliers
- Typographie en majuscules pour les titres
- Carrés/rectangles parfaits

**Interaction Philosophy** : 
- Transitions subtiles (200ms)
- Hover : changement de poids typographique
- Pas d'animations complexes

**Animation** :
- Fade-in au scroll (opacity)
- Underline animation au hover
- Transition de couleur en 150ms

**Typography System** :
- Display: IBM Plex Sans Bold 48px
- Heading: IBM Plex Sans 32px
- Body: IBM Plex Sans 16px
- Caption: IBM Plex Sans 12px

---

### Approche 2 : Moderne Chaleureux (Probabilité : 0.07)

**Design Movement** : Contemporary Minimalism avec touches organiques

**Core Principles** :
1. Minimalisme avec humanité
2. Typographie mixte (serif + sans-serif)
3. Couleurs neutres + accent subtil
4. Formes légèrement arrondies

**Color Philosophy** :
- Fond blanc cassé (#FAFAF8)
- Texte gris chaud (#2D2D2D)
- Accent terracotta (#C17A6B)
- Gris secondaire (#A0A0A0)

**Layout Paradigm** :
- Asymétrie légère
- Sections avec padding généreux
- Images intégrées naturellement
- Alternance texte/image

**Signature Elements** :
- Coins légèrement arrondis (4-8px)
- Séparateurs subtils (lignes fines)
- Icônes custom minimalistes
- Ombres très légères

**Interaction Philosophy** :
- Transitions fluides (250ms)
- Hover : légère augmentation de l'échelle (1.02)
- Focus visible mais discret

**Animation** :
- Slide-in au scroll
- Scale-up au hover
- Fade + slide combinés
- Respects prefers-reduced-motion

**Typography System** :
- Display: Lora Bold 48px
- Heading: Inter 32px
- Body: Inter 16px
- Caption: Inter 12px

---

### Approche 3 : Données Visuelles (Probabilité : 0.06)

**Design Movement** : Data-Driven Design / Information Architecture

**Core Principles** :
1. Visualisation des compétences
2. Couleurs par domaine (SIG, Data, Tools)
3. Graphiques et diagrammes intégrés
4. Hiérarchie visuelle forte

**Color Philosophy** :
- Fond blanc (#FFFFFF)
- Primaire bleu SIG (#1E40AF)
- Secondaire orange Data (#EA580C)
- Tertiaire vert Analyse (#16A34A)
- Texte gris (#374151)

**Layout Paradigm** :
- Grille 3 colonnes
- Cartes avec couleurs distinctes
- Visualisations de compétences
- Infographie intégrée

**Signature Elements** :
- Cartes colorées avec icônes
- Badges de compétences
- Graphiques de maîtrise
- Lignes de connexion

**Interaction Philosophy** :
- Hover : surélevation (shadow)
- Click : interaction avec cartes
- Transitions de couleur

**Animation** :
- Card lift au hover
- Compteurs animés
- Icônes animées
- Barre de progression animée

**Typography System** :
- Display: Poppins Bold 48px
- Heading: Poppins 32px
- Body: Open Sans 16px
- Caption: Open Sans 12px

---

## Choix Final : Approche 2 - Moderne Chaleureux

**Justification** : 
Cette approche combine le minimalisme professionnel de naledi.co.uk avec une humanité subtile. Elle est :
- Facile à maintenir pour un débutant
- Professionnelle sans être froide
- Extensible pour ajouter des projets
- Accessible et lisible
- Moderne mais intemporelle

**Palette de couleurs** :
- Fond : #FAFAF8 (blanc cassé)
- Texte principal : #2D2D2D (gris chaud)
- Texte secondaire : #666666 (gris)
- Accent : #C17A6B (terracotta)
- Bordures : #E5E5E3 (gris très clair)

**Typographie** :
- Titres : Lora (serif, chaleureux)
- Corps : Inter (sans-serif, lisible)
- Tailles : 48px (display), 32px (h1), 24px (h2), 16px (body)

**Éléments signature** :
- Coins arrondis 6px
- Ombres subtiles
- Séparateurs fins
- Espacements généreux
